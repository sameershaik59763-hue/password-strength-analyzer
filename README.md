# Password Strength Analyzer 🔐

This project is a Python-based Password Strength Analyzer that evaluates the strength of passwords using key security parameters.

## 🚀 Features
- Checks password length
- Detects uppercase and lowercase letters
- Identifies numbers and special characters
- Classifies passwords as Weak, Medium, or Strong

## 💻 Technologies Used
- Python
- Regular Expressions (re)

## 📌 Usage
Run the Python script and it will analyze the password strength.

## 🎯 Purpose
This tool helps users create secure passwords and promotes better cybersecurity practices.
